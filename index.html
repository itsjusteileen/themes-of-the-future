JS

Create a new block using InnerBlocks and templates, creating a set of blocks based off the template.


						fconst el = wp.element.createElement;
						const { registerBlockType } = wp.blocks;
						const { InnerBlocks } = wp.blockEditor;
						 
						const BLOCKS_TEMPLATE = [
							[ 'core/image', {} ],
							[ 'core/paragraph', { placeholder: 'Image Details' } ],
						];
						 
						registerBlockType( 'myplugin/template', {
							title: 'My Template Block',
							category: 'widgets',
							edit: ( props ) => {
								return el( InnerBlocks, {
									template: BLOCKS_TEMPLATE,
									templateLock: false
								});
							},
							save: ( props ) => {
								return el( InnerBlocks.Content, {} );
							},
						});

Custom Post Type

Create and register a custom post type. 


						function sidetrack_register_audio_post_type() {
							$args = array(
								'public' => true,
								'label'  => 'Audio',
								'template_lock' => all,
								'show_in_rest' => true,
								'template' => array(
									array( 'core/image', array(
										'align' => 'left',
									) ),
									array( 'core/audio', array(
										'align' => 'right',
									) ),
									array( 'core/heading', array(
										'placeholder' => 'Lyrics',
									) ),
									array( 'core/paragraph', array(
										'placeholder' => 'Add lyrics',
									) ),
								),
							);
							register_post_type( 'audio', $args );
						}
						add_action( 'init', 'sidetrack_register_audio_post_type' );

Block Based Themes

Block Based Themes

What is a Block Based Theme?

A WordPress theme with templates entirely composed of blocks so that in addition to the post content of the different post types (pages, posts, …), the block editor can also be used to edit all areas of the site: headers, footers, sidebars, etc.

Block Based Themes

Block-based Theme Structure


						theme
							|__ style.css
							|__ functions.php
							|__ block-templates
								|__ index.html
								|__ single.html
								|__ archive.html
								|__ ...
							|__ block-template-parts
								|__ header.html
								|__ footer.html
								|__ sidebar.html
								|__ ...

The difference with existing WordPress themes is that the different templates in the template hierarchy, and template parts, are block templates instead of php files.

Block Based Themes

What is a Block Template

A block template is made up of a list of blocks. Any WordPress block can be used in a template. Templates can also reuse parts of their content using “Template Parts”. For example, all the block templates can have the same header included from a separate header.html template part.

Templates CPT

If you save the templates directly from the temporary Templates admin menu, you’ll be able to override your theme’s templates. Example: By using single as the title for your template and saving it, this saved template will take precedence over your theme’s single.html file. Note that it won’t take precedence over any of your theme’s templates with higher specificity in the template hierarchy. Resolution goes from most to least specific, looking first for a CPT post and then for a theme template, at each level.
